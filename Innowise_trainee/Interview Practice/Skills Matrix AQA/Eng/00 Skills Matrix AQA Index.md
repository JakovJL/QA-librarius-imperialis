## Purpose

This folder prepares answers for the topics from the `Knowledge Scope` sheet in `Skills Matrix Automation QA.xlsx`.

Each real topic from the matrix has one separate note. Empty category rows such as `Manual testings`, `Automation testing`, and `Programming languages` are not notes.

---

## How to Use the Notes

1. Read one question without looking at the answer.
2. Give a short answer aloud in English.
3. Compare your answer with the suggested answer.
4. Add a true example from your own study or project experience.
5. Mark the topic complete only when you can answer without reading.

## Scope Rules

- The order follows the `Knowledge Scope` sheet from top to bottom.
- One matrix topic equals one note.
- Category rows without questions or skill requirements are skipped.
- Java is the selected programming language. TypeScript, JavaScript, C#, and Python are outside this preparation scope.
- English is the primary version. The Russian note must convey the same meaning.
