## Цель

Эта папка предназначена для подготовки ответов по темам из листа `Knowledge Scope` файла `Skills Matrix Automation QA.xlsx`.

Для каждой реальной темы из матрицы создаётся отдельная заметка. Пустые строки-категории `Manual testings`, `Automation testing` и `Programming languages` не являются заметками.

---

## Как работать с заметками

1. Прочитать вопрос, не заглядывая в ответ.
2. Дать короткий ответ вслух на английском языке.
3. Сравнить свой ответ с предложенным вариантом.
4. Добавить правдивый пример из своего обучения или проектного опыта.
5. Отметить тему завершённой, только когда ответ получается без чтения.

## Правила области подготовки

- Порядок полностью повторяет лист `Knowledge Scope` сверху вниз.
- Одна тема матрицы равна одной заметке.
- Строки-категории без вопросов или требований к знаниям пропускаются.
- В качестве языка программирования выбрана Java. TypeScript, JavaScript, C# и Python не входят в эту подготовку.
- Английская версия является основной. Русская заметка должна передавать тот же смысл.
