
| Элемент                  | Как написать в Markdown                          | Результат / Особенности в Obsidian                                       |
| ------------------------ | ------------------------------------------------ | ------------------------------------------------------------------------ |
| **Заголовки**            | `# H1`  <br>`## H2`  <br>`### H3`                | 6 уровней. `#` + пробел.                                                 |
| **Жирный текст**         | `**жирный**` или `__жирный__`                    | **жирный**                                                               |
| **Курсив**               | `*курсив*` или `_курсив_`                        | _курсив_                                                                 |
| **~~Зачеркнутый~~**      | `~~текст~~`                                      | ~~текст~~                                                                |
| **==Выделение==**        | `==текст==`                                      | ==Выделение цветом== (фишка Obsidian)                                    |
| **Внутренняя ссылка**    | `[[Название заметки]]`                           | Основа Obsidian. Создает связь в графе.                                  |
| **Ссылка с алиасом**     | `[[Название заметки\|Текст ссылки]]`             | Отображается как [Текст ссылки](https://.../), но ведет в ту же заметку. |
| **Внешняя ссылка**       | `[Текст](https://url.com)`                       | [Текст](https://obsidian.md/)                                            |
| **Картинка**             | `![[image.png]]`                                 | Встроит картинку из вашего хранилища.                                    |
| **Картинка из веба**     | `![Описание](https://url.com/image.jpg)`         | Подгрузит картинку из интернета.                                         |
| **Список маркированный** | `- Пункт 1`  <br>`- Вложенный пункт`             | Минус и пробел. Вложенность табуляцией.                                  |
| **Список нумерованный**  | `1. Пункт 1`  <br>`2. Пункт 2`                   | Цифра, точка и пробел.                                                   |
| **Чек-лист (Задачи)**    | `- [ ] Не выполнено`  <br>`- [x] Выполнено`      | В Obsidian можно кликать мышкой.                                         |
| **Цитата**               | `> Текст цитаты`  <br>`>> Вложенная цитата`      |                                                                          |
| **Код (внутри строки)**  | `` `код` ``                                      | `код`                                                                    |
| **Блок кода**            | ` ```python `  <br>`print("Hello")`  <br>` ``` ` | Подсветка синтаксиса по названию языка.                                  |
| **Горизонтальная линия** | `---` или `***`                                  | ---                                                                      |
| **Формула (LaTeX)**      | `$E = mc^2$`  <br>`$$ \int_a^b f(x)dx $$`        | Встроенная $E=mc^2$ или выносная.                                        |
| **Тег**                  | `#важный_тег`                                    | Для поиска и организации.                                                |


### Ссылки Obsidian (главная фишка)

- `[[Название заметки]]` — ссылка на другую заметку
    
- `[[Название заметки|Текст]]` — ссылка с отображаемым текстом
    
- `[[Название заметки#заголовок]]` — ссылка на конкретный заголовок
    
- `[[Название заметки^блок-id]]` — ссылка на конкретный абзац/блок
    
- `![[Название заметки]]` — **встроить** содержимое другой заметки прямо сюда
    
- `![[аудио.mp3]]` / `![[видео.mp4]]` / `![[pdf.pdf]]` — встроить медиа/документ
    
- `#тег` — тег (работает поиск)
    
- `#вложенный/тег` — вложенный тег
    

### 📋 Списки и задачи

- `- пункт` маркированный список
    
- `1. пункт` нумерованный
    
- `- [ ] задача` чекбокс (кликабельный)
    
- `- [x] выполнено` зачёркнутый чекбокс
    
- Табуляция `Tab` для вложенности любого списка
    

### 📊 Таблицы

text

| Заголовок 1 | Заголовок 2 |
|-------------|-------------|
| Данные      | Данные      |

Выравнивание: `:---` слева, `:---:` центр, `---:` справа

### 🧮 Математика LaTeX

- `$E=mc^2$` — формула внутри строки
    
- `$$ \int_0^\infty e^{-x^2} dx $$` — выносная формула на отдельной строке

### 📌 Frontmatter (Свойства заметки)

text

---
теги: [важно, идеи]
дата: 2026-04-12
статус: в работе
---

(пишется в самом верху заметки, Obsidian показывает это в отдельной панели свойств)

### 🎯 Эмодзи и иконки

- `Win + .` или `Ctrl + Cmd + Space` → панель эмодзи системы
    
- `:dart:` → 🎯 (требуется плагин Emoji Toolbar)
    

### 💎 Чисто Obsidian-фишки (не стандартный MD)

- `%% комментарий %%` — текст, который видно только в режиме редактирования, в режиме чтения исчезает
    
- `^id-блока` — Obsidian автоматически генерирует при выделении и нажатии `[[`
    
- Свойства `field:: value` внутри текста (Dataview-стиль, но работает и встроенный поиск)
    
- Возможность **перетаскивать** файлы мышкой в редактор — автоматически вставится `![[файл]]`
